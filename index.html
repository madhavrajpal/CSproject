ABOUT RENAME DIRS



RENAME DIRS is a python script to rename multiple directories in a uniform format.
We allow people to easily rename multiple files in a uniform manner,
and also give the option to revert back to the previous version of the file name.

WHY RENAME DIRS


We personally like our files to be arranged in a perfect and neat manner with a specific format.
But, sometimes during intense competitions, we end up saving the files in a random format.
Inspired by our personal problem, we have created RENAME DIRS to make the process quicker and easier.
